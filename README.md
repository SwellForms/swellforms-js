# @swellforms/js

Core JS SDK for SwellForms.

> ⚠ This package is under active development and is not yet stable. Use at your own risk.

## Install

```bash
npm i @swellforms/js
```

## Usage

```javascript
import { submitForm } from '@swellforms/js'

await submitForm({
    formId: 'YOUR-FORM-ID',
    name: 'Jane Doe',
    email: 'jane@example.com'
})
```
